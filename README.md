# harshh_b_arts | Professional Sketch Artist Portfolio

A high-end, responsive portfolio website for **Harshal Borase**, a professional sketch artist. This website showcases realistic charcoal and graphite artworks, anatomical sketches, and cinematic art.

## 🚀 Features

- **Multi-Page Architecture**: Separated pages for Home/About, Gallery, Services, and Contact for better performance and organization.
- **Interactive Art Gallery**: 
    - Category filtering (Portraits, Anatomical Sketches, Cinematic Art).
    - Professional "Frame" styling for artworks.
    - Custom Lightbox with Next/Prev navigation.
- **Mobile-First Design**: 
    - Fully responsive UI for all screen sizes.
    - User-friendly Hamburger menu for mobile navigation.
    - Touch-optimized buttons and forms.
- **WhatsApp Integration**: 
    - "Book Now" system that sends order details directly to WhatsApp.
    - Contact form that pre-fills WhatsApp messages for inquiries.
- **Enhanced UX**:
    - **History API Support**: Mobile back button closes the lightbox instead of exiting the site.
    - **Scroll-Lock**: Background scrolling is disabled when viewing photos.
    - **Smooth Animations**: Intersection Observer API used for "Scroll Reveal" effects.

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3 (Vanilla CSS), JavaScript (Vanilla JS)
- **Icons**: Font Awesome 6.0
- **Fonts**: Google Fonts (Playfair Display & Inter)
- **Integration**: WhatsApp Business API

## 🎨 Design Philosophy

The website uses a **Premium Dark Theme** with a **Gold (#d4af37)** accent to reflect the professional and artistic nature of the sketches. Typography is a blend of a classic Serif (`Playfair Display`) for headings and a clean Sans-Serif (`Inter`) for body text.
