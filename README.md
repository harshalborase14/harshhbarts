# harshh_b_arts | Professional Sketch Artist Portfolio

Welcome to the official portfolio of **Harshal Borase**, a professional sketch artist specializing in charcoal and graphite realism. This project is a modern, responsive, single-page website designed to showcase artistic work and streamline the commission booking process.

## ✨ Key Features

- **Elegant Dark Theme:** A premium charcoal and gold aesthetic tailored for artists.
- **Interactive Gallery:** A filtered grid for different art categories (Portraits, Anatomical, Cinematic).
- **Lightbox Viewer:** Click on any artwork to view it in full-screen with navigation controls.
- **Commission Ordering System:**
    - Dedicated pricing cards for A4 and A3 sketches.
    - Custom order modal to collect customer details.
    - **Direct WhatsApp Integration:** Automatically generates a formatted order message and redirects customers to WhatsApp.
- **Responsive Design:** Fully optimized for Desktop, Tablets, and Mobile phones.
- **Smooth Animations:** Reveal-on-scroll effects for a premium browsing experience.
- **Functional Contact Form:** Direct WhatsApp redirection for general inquiries.

## 🛠️ Technologies Used

- **HTML5:** Semantic structure for SEO and accessibility.
- **CSS3 (Custom Properties & Flexbox/Grid):** Modern styling without external frameworks.
- **JavaScript (Vanilla):** Interactive gallery filtering, lightbox logic, and modal management.
- **Google Fonts:** Inter & Playfair Display for high-end typography.
- **Font Awesome:** Professional iconography.

## 🎨 How to Use

1. **Local View:** Simply open `index.html` in any web browser.
2. **Add Your Artwork:**
   - Replace the `placeholder-img` divs in `index.html` with your actual image paths: `<img src="your-image.jpg" alt="Description">`.
   - Update the `data-category` attribute to match your art style.
3. **Customize Pricing:** Edit the pricing section in `index.html` to update your rates.

## 🌐 Deployment (Hosting)

### Via Netlify (Recommended)
1. Go to [Netlify](https://www.netlify.com/).
2. Drag and drop the `ridhi-arts-portfolio` folder into the deployment box.
3. Your site is live!

### Via GitHub Pages
1. Create a new repository on GitHub.
2. Upload all files.
3. Go to **Settings > Pages** and enable hosting for the `main` branch.
